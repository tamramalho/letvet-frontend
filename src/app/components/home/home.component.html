Olá! Sejam bem vindos ao sistema Let-Vet.
Trata-se de um sistema para gerenciamento de consultas dos seus pets, cujo objetivo é facilitar e agilizar o atendimento dos pets, com total transparência para os seus tutores.

Esse utilitário simula consultas em uma determinada clínica veterinária, desenvolvido em Angular 12 e Spring Boot 2. O mesmo está rodando na nuvem sendo o front-end no Vercel e o back-end no Heroku. O código fonte do projeto pode ser acessado clicando no link GitHub no menu lateral.
